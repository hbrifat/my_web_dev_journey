Important FAQ

What is the Document Object Model (DOM) ?

DOM represents the document (i.e web page) as nodes and objects; that way, programming languages can interact with the page. We can change structure, style and content. DOM allows us to interact with web pages dynamically.

How do you select an element from the DOM ?

We can select an element from the DOM by various methods such as getElementById(), getElementsByClassName(), getElementByTagName(), querySelector(), querySelectorAll()

What is event delegation in the context of the DOM, and why it is useful ?

Event Delegation is a pattern based on the concept of Event Bubbling. It's an event-handling pattern that allows us to handle events at a higher level in the DOM tree other than the level where the event was first received. The idea is that we "delegate" the handling of an event to a different element instead of the actual element that received the event.

How do you manipulate an element's attributes and styles using the DOM ?

The classList only provide methods to change the class of element. To change other attributes like id, href, src - we can use the setAttribute() method. This method accepts two arguments - the name of the attribute to set & the value of the attribute to set. To get attribute's value, we can use the getAttribute() method.
